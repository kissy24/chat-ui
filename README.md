# gemini-discord

Feel free to use Gemini's free slots on discord to use the generated AI.

## Usage

1. Add dotenv file.

    ```txt
    GOOGLE_API_KEY="hogehoge"
    DISCORD_TOKEN="fugafuga"
    ```

2. Run python script.

    ```bat
    uv run main.py
    ```

## Requirements

- Python 3.12
- uv
